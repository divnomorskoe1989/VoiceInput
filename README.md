# VoiceInput

> **Windows only.** Приложение работает только на Windows 10/11. macOS и Linux не поддерживаются — используются Win32 API (SendInput, clipboard, winsound, системный трей).

Десктопное приложение для голосового ввода текста на Windows. Нажми горячую клавишу — говори — текст автоматически вставляется в активное окно.

## Возможности

- **Распознавание речи в реальном времени** через [Deepgram](https://deepgram.com/) (модель Nova-2)
- **Горячая клавиша Ctrl+Win** для начала/остановки записи
- **Автоматическая вставка текста** в любое активное окно (терминал, IDE, браузер, Telegram)
- **Системный трей** — иконка показывает состояние (зелёная = готов, красная = запись)
- **Автостоп по тишине** — запись останавливается если молчишь дольше порога
- **Нормализация текста** — голосовые команды пунктуации («запятая», «точка», «вопросительный знак»)
- **Промежуточные результаты** — текст появляется по мере произнесения (опционально)
- **Автозапуск** — можно включить через меню трея

## Быстрый старт

### 1. Установка зависимостей

```powershell
pip install -r requirements.txt
```

### 2. Получение API-ключа Deepgram

1. Зарегистрируйся на [console.deepgram.com](https://console.deepgram.com/) (есть бесплатный тариф)
2. Создай API-ключ в разделе **API Keys**
3. Скопируй ключ

### 3. Настройка

Скопируй `.env.example` в `.env`:

```powershell
copy .env.example .env
```

Открой `.env` в любом текстовом редакторе и вставь свой ключ:
```ini
DEEPGRAM_API_KEY=вставь_сюда_свой_ключ
VOICEINPUT_STT_BACKEND=deepgram
```

> **Важно:** файл `.env` содержит секретный ключ — он не попадает в git (добавлен в `.gitignore`).

### 4. Запуск

```powershell
python main.py --listen-hotkey
```

Или двойной клик по `start_hidden.vbs` для запуска без окна консоли.

### 5. Использование

1. Нажми **Ctrl+Win** — прозвучит сигнал, начнётся запись
2. Говори — текст вставляется в активное окно
3. Нажми **Ctrl+Win** ещё раз — запись остановится
4. Или подожди ~5 секунд тишины — автостоп

## Конфигурация

Все параметры задаются через переменные окружения или файл `.env`:

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| `DEEPGRAM_API_KEY` | — | API-ключ Deepgram (обязателен) |
| `VOICEINPUT_STT_BACKEND` | `deepgram` | Бэкенд распознавания: `deepgram` или `mock` |
| `VOICEINPUT_HOTKEY` | `ctrl+win` | Горячая клавиша (модификаторы через `+`) |
| `VOICEINPUT_SILENCE_TIMEOUT_SEC` | `10.0` | Таймаут тишины для автостопа (секунды) |
| `VOICEINPUT_SILENCE_RMS_THRESHOLD` | `300.0` | Порог RMS для определения тишины |
| `VOICEINPUT_DEEPGRAM_MODEL` | `nova-2` | Модель Deepgram |
| `VOICEINPUT_DEEPGRAM_LANGUAGE` | `ru` | Язык распознавания |
| `VOICEINPUT_INTERIM_INSERT_ENABLED` | `true` | Вставлять промежуточные результаты по мере речи |
| `VOICEINPUT_DEEPGRAM_ENDPOINTING_MS` | `250` | Порог конца фразы (мс) |
| `VOICEINPUT_LOG_LEVEL` | `INFO` | Уровень логирования |
| `VOICEINPUT_HOTKEY_DEBOUNCE_MS` | `220` | Защита от двойного срабатывания (мс) |

Полный список параметров — в `.env.example`.

## Архитектура

```
main.py                 — точка входа, слушатель горячих клавиш
voice_input_app.py      — ядро приложения, управление сессией записи
audio_recorder.py       — захват аудио с микрофона (sounddevice)
stt_client.py           — клиент Deepgram WebSocket API
text_inserter.py        — вставка текста через clipboard/keystroke
text_normalizer.py      — нормализация транскрипта (пунктуация, термины)
keyboard_handler.py     — обработка горячих клавиш с debounce
tray_manager.py         — иконка в системном трее (pystray)
sound_notifier.py       — звуковые уведомления старт/стоп
config.py               — конфигурация из .env
contracts.py            — датаклассы (TranscriptEvent, InsertResult, SessionStats)
```

## Стратегии вставки текста

Приложение автоматически выбирает способ вставки в зависимости от активного окна:

| Тип окна | Определение | Стратегия |
|----------|------------|-----------|
| Терминал | cmd, powershell, bash | Прямой набор (`type_direct`) |
| Qt-приложения | Telegram, PyCharm, VSCode | Clipboard + Ctrl+V |
| Остальные | — | Ctrl+V, затем набор как фолбэк |

## Голосовые команды пунктуации

| Голосовая команда | Результат |
|-------------------|-----------|
| «запятая» | `,` |
| «точка» | `.` |
| «вопросительный знак» | `?` |
| «восклицательный знак» | `!` |
| «двоеточие» | `:` |
| «точка с запятой» | `;` |
| «тире» | ` - ` |

## Логи

Логи пишутся в `tmp/voice_input_recovered.log` с ротацией (100 МБ по умолчанию).

Ключевые маркеры в логах: `HOTKEY`, `STEP`, `MIC`, `DEEPGRAM`, `SOUND`, `SESSION_DIAG`.
